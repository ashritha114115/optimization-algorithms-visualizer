## Project Description

This project presents a simple **Optimization Algorithms Visualizer Website** created using HTML, CSS, and JavaScript.

The website demonstrates concepts implemented in my Optimization course assignments, including:

* Critical Point Analysis
* Pareto Front in Multi-objective Optimization
* Unconstrained Minimization
* Genetic Algorithms
* Simulated Annealing

The goal of this project is to provide a simple interface that links to all assignments and visualizes optimization concepts.
